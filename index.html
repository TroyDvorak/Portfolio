Projects

Digital Timecard

Designed, tested, and implemented a digital timecard system to replace the outdated paper-based system for a construction employer. By integrating Twilio for reporting hours via WhatsApp, I automated time entry and streamlined operational efficiency using Microsoft Excel. The new system saved employees over five hours per week through automation, significantly enhancing overall productivity and operational workflows.

Bald Man Go!

This 2D platformer project focuses on building a dynamic system where the player must keep up with a moving camera that progressively speeds up. The game uses physics-driven mechanics and collision detection to create smooth and challenging gameplay. Enemies with different behaviors and obstacles add complexity, while a customizable skill system lets players equip abilities to tackle specific challenges. Procedural level elements and careful testing ensure the game remains engaging and responsive, highlighting the technical effort behind creating fluid and scalable 2D game mechanics.
